# MindSpace | Center for Psychological Well-being

## About the Project

MindSpace is a web platform for scheduling appointments with psychologists, developed as part of a diploma thesis. The project demonstrates the ability to create a modern, user-friendly, and aesthetically pleasing psychological help service with a complete cycle: from specialist selection to online booking.

## Why It Is Needed

Creating this website solves several key problems:

- Reducing barriers — simplifying the process of finding and booking a psychologist
- Informed choice — getting to know specialists before the first meeting
- Accessibility — online format and convenient schedule
- Educational function — built-in materials on psychological self-help

## Key Features

- Appointment booking — specialist selection, format (in-person/online), time slot
- Psychologist profiles — 6 specialists with experience, approaches, and prices
- Pricing plans — trial, individual, group, diagnostic sessions
- Educational resources — articles, video courses with progress tracking, interactive workshops
- Reviews — 4.9 rating based on 257 reviews
- Responsive design — correct display on all devices
- Progress saving — LocalStorage for video lessons and tasks

## Technologies

- HTML5 — semantic structure
- CSS3 — responsive layout, animations, CSS variables
- JavaScript (ES6+) — dynamics, modal windows, sliders
- LocalStorage — user progress storage
- Font Awesome / Google Fonts — icons and fonts
- GitHub Pages — hosting

## What I Wanted to Demonstrate by Creating This Project

1. Proficiency in modern frontend development — clean structure, CSS/JS separation, responsiveness
2. Understanding of user experience (UX) — intuitive navigation, clear forms, visual feedback
3. Ability to create a full-fledged service — from interface to interactivity (lesson progress, time selection, modal windows)
4. Care for psychological well-being — educational materials as part of the service
5. Readiness for further development — the structure allows easy addition of new psychologists, articles, and courses
